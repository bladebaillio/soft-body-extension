# Soft Body: Modify

Use these blocks to tune physics and reposition segments.

## Physics tuning
- `set ... damping`
- `set ... spring stiffness`
- `set ... segment max velocity`
- `set ... max stretch`
- `set ... gravity strength`

## Segment behavior
- `set segment ... gravity ...`
- `set segment ... position ...`
- `set ... movement locked ...`
- `set ... segment direction ...`
- `add ... to ...`
- `remove segment ... from ...`

## Placement and style
- `place ... at x y`
- `place ... from x1 y1 to x2 y2`
- `draw lines between segments ...`
- `fill area between segments ...`
